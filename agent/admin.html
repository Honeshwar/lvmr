LVMR Pricing Admin

Update room prices using this simple editor. Enter only the new price numbers, then download the file and save it as agent/calculator-prices.json in your branch.

What each field means

Location = which property the price belongs to.

Room type = the category of room or bed.

Plan = the booking length: monthly, weekly, or two-week.

Persons = the number of people sharing the room.

  • Use the field label to find the exact price item.
  • Do not change the field names or text, only change the number.
  1. Find the location, room type, plan, and person count.
  2. Type the new price amount in the field.
  3. Click Update prices.
  4. Wait about 30 seconds for live changes to appear.
